# Power-Manager - A Web Application for Power Management

Power-Manager is a web application designed to manage power-related information, such as bills, notifications, analytics, and logs. The integration with Google Sheets suggests a dynamic way to handle and display data.

Website Link: https://max-eee.github.io/Power-Manager/
